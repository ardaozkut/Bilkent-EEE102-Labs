This file is a VIVADO Project folder that you can download to include in your projects.

The Pin configuration is as follows:

                _________
SC10 ----- SIOC|        |SIOD ----- JC4
SC9  ---- VSYNC|        |HREF ----- JC3      
SC8  ----- PCLK|        |XCLK ----- JC2
SC7  ------- D7|        |D6 ------- JC1
JB10 ------- D5|        |D4 ------- JB4
JB9  ------- D3|        |D3 ------- JB3
JB8  ------- D1|        |D0 ------- JB2
JB7  ---- RESET|________|PWDN ----- JB1
